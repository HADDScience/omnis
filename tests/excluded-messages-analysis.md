## 사유 정의

| 사유 | 설명 | 예시 |
|------|------|------|
| 암묵적 지시 | 명시적 `@멘션 + 지시`가 아닌 이름 호칭이나 맥락상 지시 | "우창님 이거 해주세요" |
| 완료 확인 불가 | 지시는 있으나 대화 내에서 완료 보고 메시지가 없음 | 지시만 있고 응답 없음 |
| 파일만 전달 | 업무 맥락 설명 없이 파일만 공유 | "File: xxx.pdf" |
| 논의/질문 | 업무 지시가 아닌 질의응답/논의 | "~하면 될까요?" |
| 일정/알림 | 미팅, 교육, 행사 등 일정 공유 | 기념식 초청장, 교육안내 |
| 중복 대화 | 두 채팅방에서 같은 업무가 논의됨 | 인턴방+수원대방 동시 전달 |
